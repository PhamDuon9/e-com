# Thế Giới SportLux — Showcase

Next.js + TypeScript + Tailwind demo storefront for badminton products.

Quick setup

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

Sanity

- Create a Sanity project and set `NEXT_PUBLIC_SANITY_PROJECT_ID` and `NEXT_PUBLIC_SANITY_DATASET` in your environment.
- Schemas are in the `sanity/schemas` folder.

Deploy

- Deploy to Vercel and set env vars for Sanity.
